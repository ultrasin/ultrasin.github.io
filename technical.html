C

The most used low programming level in college: C. This language is often taught as the first programming language in teaching institutions because it is low level language. The syntax looks like this:

#include<stdio.h>
main()
{
printf("Hello World");
}

C++

The difference with C, aside new libraries - is that introduces OOP which means Object Oriented Programming. With pointers and classes, brings a whole new functionality for modern application development.

#include <iostream>
int main()
{
std::cout << "Hello, world!\n";
}

Java

Uses modular libraries, meaning you will only need to download your .jar files and tell the program to include them for later packaging. This language also is scalable.

public class HelloWorld {
public static void main(String[] args) {
System.out.println("Hello, World!");
}
}

Eclipse and Netbeans are some IDE that will help you build your first application.

JavaScript

What is JavaScript? Well, JavaScript let you add complex features on web pages without needing anything more than just your browser. It can be programmed in any text processor.

<script language="JavaScript">
document.write('Hello, World!');
</script>

Easily creates media dynamically, animate images and more.
